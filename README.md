## 💻 How To Install?
- Clone This Repo
```
git clone https://github.com/therevolt/Zwallet-Frontend
```
- Go To Folder Repo
```
cd Zwallet-Frontend
```
- Open Command/Terminal
- Type ``` npm run dev``` To Start Website
